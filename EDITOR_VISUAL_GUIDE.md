# Guia Completo do Editor Visual de Landing Page

## Visão Geral

O novo Editor Visual de Landing Page é um construtor visual completo que permite personalizar totalmente cada seção da página inicial sem necessidade de modificar código. Com mais de 100 elementos minimalistas, 20+ fontes Google, animações fluidas e pré-visualização em tempo real, você pode criar landing pages profissionais e responsivas.

## Funcionalidades Principais

### 1. **Gerenciamento de Seções**

O editor permite criar, editar, duplicar e remover seções de forma intuitiva.

#### Criar Nova Seção

1. Na sidebar esquerda, clique no botão **"+ Nova"**
2. A nova seção será criada com o título padrão "Nova Secção"
3. Selecione a seção para começar a editar

#### Configurar Seção

Cada seção possui as seguintes configurações:

| Configuração | Descrição | Valores |
|:-------------|:----------|:--------|
| **Título** | Nome da seção | Texto livre |
| **Altura** | Altura da seção em pixels ou viewport height | ex: `500px`, `100vh` |
| **Cor de Fundo** | Cor de fundo da seção | Padrão, Card, Primária, Muted, Preto |
| **Ativo** | Visibilidade da seção na landing page | Ativado/Desativado |

#### Reordenar Seções

Use os botões de seta (↑/↓) na sidebar para mover seções para cima ou para baixo. A ordem é atualizada automaticamente.

#### Duplicar Seção

Clique no botão **"Duplicar"** para criar uma cópia da seção atual com todos os seus elementos.

#### Remover Seção

Clique no botão **"Remover"** para deletar a seção (esta ação não pode ser desfeita).

### 3. **Posicionamento e Redimensionamento**

Todos os elementos podem ser posicionados e redimensionados com precisão.

#### Drag-and-Drop

1. Clique e mantenha pressionado um elemento na canvas
2. Arraste para a posição desejada
3. Solte para confirmar

#### Controles de Posição

Na aba **"Posição"** do painel de propriedades:

- **Posição Horizontal (X)**: 0-100% da largura da seção
- **Posição Vertical (Y)**: 0-100% da altura da seção
- **Largura**: 0-100% da largura da seção
- **Altura**: 0-100% da altura da seção (para imagens e vídeos)
- **Z-Index**: Ordem de sobreposição dos elementos

Use os sliders para ajuste fino ou digite valores diretamente.

#### Personalizar Texto

Para elementos de texto, customize:

- **Fonte**: Selecione entre 20+ fontes Google
- **Tamanho**: 8px a 120px
- **Peso**: Leve (300), Normal (400), Médio (500), Semi-negrito (600), Negrito (700), Extra-negrito (800), Preto (900)
- **Alinhamento**: Esquerda, Centro, Direita
- **Cor**: Escolha qualquer cor com o seletor de cores
- **Espaçamento entre letras**: Ajuste fino do kerning
- **Altura de linha**: Multiplicador de altura de linha (1.5, 2, etc.)

### 5. **Estilo e Cores**

Customize cores, bordas, sombras e efeitos de cada elemento.

#### Cores

- **Cor do Texto**: Use o seletor de cores ou digite o valor hex/rgb
- **Cor de Fundo**: Personalize o fundo de qualquer elemento
- **Cor da Borda**: Defina a cor das bordas

#### Bordas

- **Largura da Borda**: 0-20px
- **Raio da Borda**: Crie cantos arredondados (0-50px)
- **Estilo da Borda**: Sólido, Tracejado, Pontilhado

#### Efeitos

- **Opacidade**: 0-100% (controle de transparência)
- **Sombra**: Adicione sombras aos elementos
- **Filtros**: Aplique filtros CSS (blur, brightness, etc.)

### 6. **Animações Fluidas**

Escolha entre 15+ animações de entrada com controle total sobre timing.

#### Tipos de Animação

| Animação | Descrição |
|:---------|:----------|
| **Nenhuma** | Sem animação |
| **Surgir (Cima)** | Fade-up: elemento aparece subindo |
| **Surgir (Baixo)** | Fade-down: elemento aparece descendo |
| **Surgir (Esquerda)** | Fade-left: elemento aparece da esquerda |
| **Surgir (Direita)** | Fade-right: elemento aparece da direita |
| **Aparecer Suave** | Fade-in: fade simples |
| **Zoom In** | Zoom de entrada |
| **Zoom Out** | Zoom inverso |
| **Deslizar Esquerda** | Slide-left: desliza da esquerda |
| **Deslizar Direita** | Slide-right: desliza da direita |
| **Deslizar Cima** | Slide-up: desliza de cima |
| **Deslizar Baixo** | Slide-down: desliza de baixo |
| **Bounce** | Efeito de salto |
| **Flip** | Rotação 3D |
| **Rotar** | Rotação contínua |
| **Pulsar** | Efeito de pulsação |

#### Controlar Animações

- **Tipo de Animação**: Escolha entre 15+ opções
- **Easing**: Linear, Ease, Ease-in, Ease-out, Ease-in-out
- **Duração**: 0.1s a 5s
- **Delay**: Atraso antes da animação começar (0s a 5s)
- **Repetições**: Número de vezes que a animação se repete (-1 para infinito)

### 7. **Botões com Links**

Crie botões interativos com links personalizáveis.

#### Configurar Botão

1. Adicione um elemento de **Botão** da biblioteca
2. Na aba **"Propriedades"**, configure:
   - **Texto do Botão**: Texto exibido no botão
   - **Link**: URL para onde o botão aponta
   - **Abrir em**: Mesma aba, nova aba, janela pai, ou topo

#### Personalizar Aparência

- **Cor de Fundo**: Cor do botão
- **Cor do Texto**: Cor do texto do botão
- **Raio da Borda**: Crie botões arredondados
- **Tamanho da Fonte**: Ajuste o tamanho do texto

### 8. **Imagens e Vídeos**

Adicione e personalize imagens e vídeos com controles avançados.

#### Adicionar Imagem

1. Clique em **"Carregar Imagem"** no painel de propriedades
2. Selecione um arquivo do seu computador
3. A imagem será enviada para o Supabase Storage
4. Ajuste posição, tamanho e estilo

#### Adicionar Vídeo

1. Adicione um elemento de **Vídeo**
2. Cole a URL do vídeo (YouTube ou Vimeo)
3. O vídeo será incorporado automaticamente

#### Controles de Imagem

- **Object-Fit**: Cover (preenche), Contain (cabe inteiro), Fill, Scale-down
- **Posição**: X, Y, Largura, Altura
- **Efeitos**: Opacidade, Sombra, Filtros

### 9. **Pré-visualização em Tempo Real**

O editor oferece dois modos de visualização:

#### Modo Edição

- Veja os elementos com controles de edição
- Arraste, redimensione e personalize
- Visualize as mudanças instantaneamente

#### Modo Preview

- Clique em **"Preview"** no topo
- Veja como a seção aparecerá na landing page
- Sem controles de edição, apenas visualização

### 10. **Salvamento Automático**

As mudanças são salvas no Supabase quando você clica em **"Salvar"**. Todas as seções, blocos e propriedades são persistidas no banco de dados.

## Fluxo de Trabalho Recomendado

1. **Criar Seção**: Clique em "+ Nova" para criar uma nova seção
2. **Configurar Seção**: Defina título, altura e cor de fundo
3. **Adicionar Elementos**: Use a biblioteca para adicionar elementos
4. **Posicionar**: Arraste os elementos para as posições desejadas
5. **Personalizar**: Use o painel de propriedades para ajustar cores, fontes, animações
6. **Visualizar**: Clique em "Preview" para ver o resultado final
7. **Salvar**: Clique em "Salvar" para persistir as mudanças
8. **Publicar**: A seção aparecerá na landing page se estiver marcada como "Ativa"

## Dicas e Truques

### Organização

- Use títulos descritivos para as seções (ex: "Hero", "Funcionalidades", "Sobre")
- Agrupe elementos relacionados na mesma seção
- Use a reordenação para criar um fluxo lógico

### Design

- Mantenha consistência com as fontes (máximo 3 fontes por página)
- Use cores complementares do tema
- Deixe espaço em branco para melhor legibilidade
- Teste em diferentes tamanhos de tela

### Performance

- Comprima imagens antes de fazer upload
- Use formatos modernos (WebP, AVIF)
- Limite o número de animações simultâneas
- Use delays para criar efeito cascata

### Acessibilidade

- Sempre adicione texto alternativo para imagens
- Use contraste suficiente entre texto e fundo
- Certifique-se de que os botões são clicáveis em mobile
- Teste com leitores de tela

## Resolução de Problemas

### Imagem não aparece após upload

- Verifique se o arquivo é uma imagem válida (JPG, PNG, WebP, GIF)
- Verifique a conexão com a internet
- Tente fazer refresh da página
- Verifique se o Supabase Storage está acessível

### Alterações não são salvas

- Certifique-se de clicar em **"Salvar"**
- Verifique se está logado como admin
- Verifique a conexão com o Supabase
- Verifique o console do navegador para erros

### Elemento não aparece na canvas

- Verifique se o elemento está dentro dos limites da seção (0-100% em X e Y)
- Verifique se o Z-Index está correto
- Verifique se a opacidade não está em 0%
- Tente aumentar a altura do elemento

### Animação não funciona

- Certifique-se de que a animação não está definida como "Nenhuma"
- Verifique se a duração é maior que 0
- Tente recarregar a página
- Verifique se o navegador suporta animações CSS
